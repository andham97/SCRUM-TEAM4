(route) New task to group task list

Route:
MethodPath
POST /api/tasks
Add new task to the group task list. The task will be made a repeating if time_interval is specified and > 0. It will repeat until the date specified as deadline.
Body Parameters:
Name Type Description
group_id number the groups id
todo_text string what the task involves
datetime_deadline number set deadline for task (Optional)
datetime_done number when task is done (Optional)
time_interval number number of days
